# Older Sibling

> This is a short message generator, as part of the Full-Stack course on Codecademy.

## Overview

> The program generates a message with a predetermined structure, simulating and older sibling waking a younger sibling, demanding a chore completed with an additional comment.

> The initial greeting (insult), required chore and final comment are randomly selected from seperate arrays and returned back to the user. Additional inputs can be used by following the commented instruction.

## Install/Run

> Download this repository.

> Use [Node.js](https://nodejs.org/en/) to run
